File Sharing

Introduction History Methods Impact



Common methods of File Sharing today are primarily through World Wide Web-based hyperlinked documents and the use of distributed peer-to-peer (P2P) networking. Through the years, many software applications and networks arose to aid in the sharing of electronic files. Some of these popular networks include:

  1. Gnutella
  2. eDonkey2000
  3. the now-defunct Kazaa network
  4. BitTorrent
