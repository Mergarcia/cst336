File Sharing

Introduction History Methods Impact



Early File Sharing came in the form of "Manual Sharing" using removable media like floppy disks from the 1980's to USB's. With the advent of the World Wide Web in the 1990's and centralized computer file server installations on computer networks, has allowed file sharing to grow on a massive scale. The increasing popularity of mp3 music format in the late 1990s led to the release and growth of Napster as well as many other applications that are solely based on file sharing.
